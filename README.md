# Oumaima Toufali - Portfolio

A premium, interactive AI and Data Science portfolio built with Next.js 16, TypeScript, Tailwind CSS, and Framer Motion. 
Features smooth animations, modern glassmorphism design, and showcases impactful projects in AI, LLMs, and Data Engineering.

## Technologies Used
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons & Lucide React

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
